TyperPro is a high-performance desktop application designed to measure, analyze, and improve typing proficiency. Built using **C#**, **Avalonia UI**, and the **MVVM** architectural pattern, it provides a seamless, cross-platform experience for participants at Cybrella 2026.

---

## Key Features
* **Real-time Analytics:** Watch your WPM and Accuracy update dynamically as you type.
* **MVVM Architecture:** Clean separation of concerns for high maintainability.
* **Visual Feedback:** Instant UI highlighting for correct and incorrect keystrokes.
* **Customizable Challenges:** Supports various difficulty levels and custom text snippets.
* **Cross-Platform:** Native performance on Windows, macOS, and Linux.

---

## Tech Stack
* **UI Framework:** [Avalonia UI](https://avaloniaui.net/)
* **Logic:** C# / .NET 8.0+
* **Pattern:** Model-View-ViewModel (MVVM)
* **Library:** `CommunityToolkit.Mvvm` for messaging and observables.
